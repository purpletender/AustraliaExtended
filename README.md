# AustraliaExtended-dataset
A custom profile dataset for vatSys based on the default Australia dataset  
*This dataset includes functionality for the vatSys profile updater*  

## Installation
Download the [latest release](https://github.com/purpletender/AustraliaExtended/releases)  
Extract the downloaded ZIP file and copy the `AustraliaExtended` folder into `Documents/vatSys Files/Profiles`  
Hold **SHIFT** whilst vatSys is opening (or once vatSys has launched navigate to *Settings -> Change Profile*) then select the `Australia Extended` profile  

## Updates
Updates made in the offical Australia dataset, will be brought across as soon as practical  
When an update to this profile is made, vatSys will prompt you to download and replace the existing profile
